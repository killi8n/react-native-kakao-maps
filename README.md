# react-native-kakao-maps

kakao maps for react native

## Installation

```sh
npm install react-native-kakao-maps
```

## Usage

```js
import KakaoMaps from "react-native-kakao-maps";

// ...

const result = await KakaoMaps.multiply(3, 7);
```
